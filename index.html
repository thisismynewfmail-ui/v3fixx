§ 3CONTEXT & ROLLING WINDOW
Max consecutive tool rounds before the agent must finalise a text reply.
◢ ROLLING CONTEXT WINDOW · When token usage crosses the threshold, the oldest middle turns "fall off" until usage drops below the limit. The system prompt and the first user message are always preserved (to keep the original task intact); the most recent turns are preserved too. Tool-call / tool-result pairs are removed together so the trace stays coherent. Indicator: CTX meter in the status bar.
